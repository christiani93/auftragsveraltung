Hilfe & Übersicht

🐞 Bug / Wunsch melden
Kurz gesagt: Diese App verwaltet Kunden, ihre Anlagen/Anlagenteile, daraus entstehende Aufträge, die Zeit-Erfassung darauf (Stempeluhr + manuelle Buchungen), Messprotokolle für die Kontrolle sowie Revisionen (z.B. Betriebsferien) als Bündelung mehrerer Aufträge.

Rollen

  • Admin — sieht alles, kann Benutzer anlegen/verwalten, Aufträge löschen, Ersatzpasswörter vergeben, beim Stempeln für andere stempeln.
  • Projektleiter — sieht alle Aufträge, kann Mitarbeiter Aufträgen zuweisen.
  • Monteur — sieht nur Aufträge die ihm zugewiesen sind oder unzugewiesen, plus alle Aufträge der Revisionen, in denen er als Mitarbeiter eingetragen ist.

Passwort + Login

  • Login mit Username (Kurzform, z.B. Mueller) — angezeigt wird überall der Anzeigename.
  • Passwort selbst ändern: 👤 Dropdown → Mein Profil.
  • Admin kann ein Ersatzpasswort vergeben (Benutzer-Verwaltung). Der User muss es beim nächsten Login zwingend ändern.
  • Neue User müssen ihr Initial-Passwort beim ersten Login ebenfalls ändern (außer der Admin schaltet das beim Anlegen explizit aus).

Kunden & Kontaktpersonen

  • Stammkunde-Häkchen → erscheint im Dashboard, hat Kontroll-Intervall in Monaten.
  • Pro Kunde mehrere Kontaktpersonen (Name, Funktion, Telefon, E-Mail). Beim Auftrag-Anlegen kann eine Kontaktperson als „Erteilt von" + Telefon übernommen werden.
  • Anlagen und Anlagenteile (Verteilung, Stromkreise) werden pro Kunde gepflegt.

Aufträge

  • Status: Offen · In Arbeit · Erledigt · Abgerechnet (Archiv, in Liste ausgeblendet).
  • Zugewiesen an — Monteur sieht den Auftrag nur wenn er zugewiesen ist oder das Feld leer ist.
  • Zu erledigen bis (Frist) — wird im Detail rot markiert wenn überfällig.
  • Termin — konkreter Vor-Ort-Termin (Datum + Uhrzeit).
  • Mehrere Bilder hochladen — werden automatisch auf 1920 px / JPEG verkleinert (auch iPhone-HEIC).

Revisionen (Großaufträge)

  • Pro Kunde planbar — typisch Betriebsferien-Zeitraum. Mit ToDo-Liste und Liste der zugeordneten Aufträge.
  • Beteiligte Mitarbeiter sehen alle Aufträge der Revision, auch wenn sie ihnen nicht direkt zugewiesen sind.
  • Aufträge mit Revision-Zuordnung werden aus der Hauptliste ausgeblendet (nur in der Revision sichtbar) — Toggle „Revisions-Aufträge anzeigen" macht sie kurzzeitig sichtbar.
  • Default-Titel bei Neu: „Revision <Jahr>".

Zeiterfassung & Stempeluhr

  • Einstempeln auf der Zeiterfassungs-Seite (auch ohne Auftrag möglich, kann nachträglich zugeordnet werden) oder direkt auf einem Auftrag-Detail.
  • Auftrag wechseln beendet die aktuelle Buchung und startet sofort eine neue auf dem neuen Auftrag.
  • Ausstempeln bucht die Zeit als Zeitbuchung am Auftrag.
  • Pause nachträglich einfügen: in der Zeitbuchungs-Tabelle pro Buchung Von/Bis eingeben — die Dauer wird automatisch reduziert.
  • Admin kann mit „Stempeln für: [User]" für einen anderen Mitarbeiter stempeln.
  • Tagesblick gruppiert pro Mitarbeiter → pro Kunde (mit Block-Zeit) → pro Auftrag.

Messprotokolle (NIN/SiNa)

  • Pro Anlage + optional pro Anlagenteil können Messprotokolle erfasst werden (Auslösezeit, R Iso, Ik, Drehrichtung etc.).
  • Messgeräte werden separat verwaltet (Kalibrierdaten). User sieht nur eigene Geräte, Admin sieht alle.
  • PDF-Export pro Protokoll + pro Kunde (alle Aufträge mit Zeitbuchungen).

Kontroll-Übersicht

  • Dashboard zeigt Stammkunden mit offenen Kontrollen / Mängeln — sortiert nach Dringlichkeit.
  • Pro Stammkunde kompakte Übersicht für den Elektrokontrolleur-Besuch.
